Git is a version control system used to track changes in files. GitHub is an online platform that hosts Git repositories and enables collaboration.

The terminal allows users to interact with the operating system using commands to create, manage, and manipulate files and run programs efficiently.

git add stages changes so they are ready to be included in the next commit.

git commit saves the staged changes as a snapshot in the project's history.

Edit files
↓
git status
↓
git add .
↓
git commit -m "  "
↓
git push origin main
